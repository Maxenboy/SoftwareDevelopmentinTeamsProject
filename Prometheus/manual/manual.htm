Manual

Produkten kommer som två program och en manual, detta dokument. De två programmen är ett registrerarprogram respektive sorteringsprogram. Dessa kan användas separat och samtidigt.

Installation

Krav

För att produkten ska kunna köras finns följande krav på datorn:

  • Java version 7, Länk till nedladdning
  • Korrekt formaterad namnfil

Format på namnfil

Första raden i filen skall vara: StartNr; Namn. Varje deltagare i namnfilen finns på var sin rad med formatet: startNr; Namn där startNr är ett heltal och Namn är godtycklig text. Det får inte finnas dubletter av startNr i filen.
Exempel på namnfil:

StartNr; Namn
1; Anders Asson
2; Bengt Bsson
3; Chris Csson
4; David Dsson
5; Erik Esson

Starta registreringsprogrammet

Ta upp ett terminalfönster och navigera till register-programmet. Kör det sedan genom att skriva följande: 

java -jar Register.jar

Starta sorteringsprogrammet

För att kunna använda sorteringsprogrammet måste du först ha kört registreringsprogammet.
Öppna kommandotolken och navigera (Windows, Linux/Mac) till mappen där programmet finns.

Marathonlopp

Skriv 
java -jar Sorter.jar resultatfil starttidsfil måltidsfil namnfil
där du byter ut starttidsfil, måltidsfil till de filer du fick från registreringsprogrammet samt skriv in din resultatfil och namnfil. Glöm inte filändelsen (.txt)!

Varvlopp

Skriv 
java -jar Sorter.jar resultatfil starttidsfil måltidsfil namnfil X
där du byter ut starttidsfil, måltidsfil till de filer du fick från registreringsprogrammet samt skriv in din resultatfil och namnfil. Glöm inte filändelsen (.txt)! X byts ut mot ett heltal, större än 0, som avgör övre gräns för antal utskrivna varv.

En resultatfil kommer att skapas utifrån det du skrev i kommandoraden.

Registrera start- eller måltider

Starta registreringsprogrammet. Skriv in tävlandes id-nummer samt välj om det är startregistrering eller målregistrering. Tryck på Register för att registrera. Om flera start- eller måltider registreras kommer detta att skrivas ut som en varning i resultatfilen, men alla registreras.

Generera resultatlista

Resultatlistan sparas alltid automatiskt i resultatfilen från registreringsprogrammet. Filen hittar du i samma mapp som programmet ligger i.
